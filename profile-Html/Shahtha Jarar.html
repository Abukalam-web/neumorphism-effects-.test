জমা
আপনার মোট জমার পরিমাণ
মিলে জমা = ২০০ টাঁকা

বিল বাকী আছে -
---
খরচ
আপনার মোট মিল = ২১ এবং মিলরেট ৩৬.২০ টাঁকা ।
আপনার খাবারের মোট বিল = ( ২১ * ৩৬.২০ )
= ৭৬১ টাঁকা মাত্র ।
সকল বিলের হিসাব -
  • গ্যাস বিল = ১২৫ টাঁকা
  • পানি বিল = ৩৮ টাঁকা
  • ওয়াইফই বিল = ৬৬ টাঁকা
  • বুয়া বিল = ৩০০ টাঁকা

আপনার মোট বিল = ৫২৯ টাঁকা
মোট খরচ = ( ৫২৯ + ৭৬১ )
= ১২৯০
